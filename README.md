# Advanced Network Scanner

A Python-based Advanced Network Scanner designed for cybersecurity professionals and students.

## Features

- Multi-threaded Host Discovery
- ARP-based Asset Discovery
- Port Scanning
- Banner Grabbing
- OS Fingerprinting
- SMB Version Detection
- SSL/TLS Analysis
- Vulnerability Assessment
- CVE Correlation
- Risk Scoring
- Reverse DNS Lookup
- Vendor Identification
- Network Topology Mapping
- CSV Report Export

## Usage

Scan a network:

```bash
python advanced_network_scanner.py 192.168.10.0/24
```

Scan specific ports:

```bash
python advanced_network_scanner.py 192.168.10.0/24 -p 22,80,443
```

Export results:

```bash
python advanced_network_scanner.py 192.168.10.0/24 -o report.csv
```
